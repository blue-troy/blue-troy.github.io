java获取根目录

2018-12-26

编译class文件并运行时输出

打包成可自行jar并运行时输出

可以总结出
System.getProperty(“java.class.path”)获取的是不解析jar包情况下的class位置
new File(“.”) 获取到的是执行命令的当前目录 相当于 pwd命令
App.class.getClassLoader().getResource(“”).getPath() = App.class.getResource(“/“).getPath() 获取的是解析jar包情况下的class地址。即地址中包括class在jar包中的位置
